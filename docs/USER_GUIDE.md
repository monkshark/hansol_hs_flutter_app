# 사용자 가이드

> English: [USER_GUIDE_en.md](./USER_GUIDE_en.md)

한솔고등학교 앱 사용법을 재학생·교사·졸업생·학부모 각 역할별로 정리합니다. 처음 설치한 분은 "시작하기"부터 보시고, 개별 기능만 궁금하면 "주요 기능" 섹션으로 바로 가셔도 됩니다.

## 시작하기

### 1. 앱 설치
- Android: Play Store에서 "한솔고등학교" 검색
- iOS: App Store에서 "한솔고등학교" 검색
- 최소 버전 미만은 실행 시 업데이트 다이얼로그가 뜹니다

### 2. 로그인
4가지 중 편한 방법으로:
- Google
- Apple (iOS)
- Kakao
- GitHub

비밀번호를 만들지 않습니다. 소셜 계정만 있으면 됩니다.

### 3. 신분 선택 및 프로필 입력
- 재학생 / 졸업생 / 교사 / 학부모 중 선택
- 재학생/교사는 학년/반/번호 입력
- 이름은 실명 권장 (학교 맥락상)

### 4. 승인 대기
관리자 승인 전까지 대부분 기능이 잠깁니다. 보통 1일 이내 처리됩니다.

## 주요 기능

### 홈
- 오늘의 급식, 시간표, 캘린더를 한 화면에
- 위로 스크롤하여 D-day / 개인일정 확인
- 오른쪽 위 알림 아이콘에서 인앱 알림 보기

### 급식
- 조식 / 중식 / 석식 개별 탭
- 메뉴 탭 → 영양 성분 + 알레르기 표시
- 이미지로 저장하거나 카카오톡 등 외부 공유

### 시간표
- 내 반 또는 선택과목 기반
- 현재 교시가 자동 하이라이트
- 과목별 색상은 장기 탭하여 커스터마이징

### 일정
- 월간 캘린더 좌우 스와이프로 이동
- 빈 날짜 탭 → 개인 일정 추가
- 연속 일정은 하나의 컬러 바로 표시

### 게시판
- 6개 카테고리 + 인기글 탭
- 글쓰기 시 이미지/투표/일정 첨부 가능
- 익명 글: 익명 번호가 글마다 부여됨
- 검색창에서 최근 검색어 + n-gram 검색

### 채팅
- 우측 상단 "새 채팅" → 유저 검색
- 메시지 길게 누르면 삭제 메뉴
- 안 읽은 메시지는 뱃지로 표시

### 성적 관리
- 수시/정시 탭
- 과목별 점수 입력 → 자동 등급 / 백분위 변환
- 추이 그래프는 등급/원점수/백분위/표준점수 중 선택
- **성적은 이 기기에만 저장됩니다.** 기기 변경 시 이관 안 됨

### 개인일정 & D-day
- 6가지 기본 색 + 원형 컬러피커로 자유 선택
- D-day는 홈 화면에 핀 고정 가능

### 알림 설정
5가지 카테고리 개별 on/off:
- 댓글 / 답글
- 채팅
- 새 글 (관심 카테고리)
- 계정 (승인/정지/역할 변경)
- 급식 (로컬 알림, 시간 설정)

### 홈 화면 위젯
- **Android**: 4×2 급식 / 3×2 시간표 / 5×2 통합
- **iOS**: Medium 급식 / Medium 시간표 / Large 통합
- 자정에 자동 갱신, 앱 실행 시에도 갱신

## 관리자용

권한이 부여된 역할은 다음 기능을 추가로 사용 가능:
- **moderator** — 신고 처리, 게시글/댓글 숨김 (정지 권한 없음)
- **auditor** — 모든 신고/로그/통계 읽기 전용 (교사 감사용)
- **manager** — 정지, 공지 고정, 사용자 승인
- **admin** — 모든 권한 + 다른 사용자의 권한 변경

도구:
- **Flutter Admin 화면**: 설정 → 관리자
- **Admin Web**: 별도 웹 대시보드 주소 (관리자에게 안내) — 사이드바 메뉴는 권한별로 자동 필터링됨

상세는 [docs/features/admin-features.md](https://monkshark.github.io/hansol_hs_flutter_app/#features/admin-features.md) 참조.

## 계정 관리

### 정지 이력 확인
- 정지 중: 글/댓글/채팅 작성 불가, 조회는 가능
- 만료 자동 해제 (매시간 체크)
- 해제되면 푸시 알림 수신

### 역할 변경
- 학생회 / 방송부 등 권한이 필요하면 관리자에게 요청
- 승급 시 admin이 `moderator` / `auditor` / `manager` / `admin` 중 하나로 변경
- 변경 직후에는 앱이 자동으로 ID 토큰을 갱신해 새 권한이 즉시 반영됨

### 새 학년 업데이트
- 재학생/교사는 3월에 팝업으로 새 학년/반/번호 입력 유도
- 신분(재학생/교사 등)은 본인이 변경 불가 (관리자만)

### 탈퇴
- 설정 → 계정 → 회원 탈퇴
- 이중 확인 후 모든 데이터 즉시 파기:
  - 프로필 / 글 / 댓글 / 채팅 / 로컬 성적 / 프로필 사진
- 탈퇴 후 같은 소셜 계정으로 재가입 시 새 사용자로 처리됨

## 자주 묻는 질문 (FAQ)

**Q. 성적이 다른 기기에서도 보이나요?**
A. 아닙니다. 성적은 이 기기의 OS 암호화 저장소에만 저장되어 서버로 올라가지 않습니다. 기기 변경 시 다시 입력해야 합니다.

**Q. 카카오로 로그인했는데 프로필 사진이 달라요.**
A. 카카오 프로필 사진을 그대로 가져옵니다. 바꾸려면 설정 → 내 계정 → 프로필 사진에서 교체하세요.

**Q. 알림이 안 와요.**
A. 설정 → 알림에서 원하는 카테고리가 켜져 있는지, 시스템 앱 알림 권한이 허용되어 있는지 확인하세요. 정지 기간 중에는 일부 알림이 제한됩니다.

**Q. 오프라인인데 급식/시간표가 보이나요?**
A. 이전에 조회한 데이터는 캐시로 남아 있어 오프라인에서도 보입니다. 오프라인 중에는 상단에 빨간 배너가 표시됩니다.

**Q. 위젯이 업데이트가 안 돼요.**
A. 앱을 한 번 실행해보세요. 자동 갱신은 자정 + 1시간(iOS) / AlarmManager(Android) 주기입니다.

**Q. 게시글을 올렸는데 공지로 고정할 수 있나요?**
A. 공지 고정은 manager/admin 권한만 가능합니다. moderator는 신고 처리/숨김까지만, auditor는 읽기 전용입니다.

**Q. 정지 같은 처리에 이의를 제기하고 싶어요.**
A. 설정 → 이의신청에서 90일 이내에 사유를 적어 제출할 수 있습니다 (PIPA 권리). 관리자가 검토하면 결과가 푸시 알림으로 옵니다.

**Q. 내 데이터를 한 번에 받아볼 수 있나요?**
A. 설정 → 데이터 요청에서 신청하면 게시글/댓글/신고 이력/채팅이 JSON으로 묶여 다운로드 링크가 이메일로 옵니다. 링크는 30일 후 자동 만료됩니다.

**Q. 익명 게시글에서 누가 썼는지 알 수 있나요?**
A. 일반 사용자는 익명 번호만 보입니다. admin은 감사 로그로 실명을 확인할 수 있으나, 모든 조회가 `admin_logs`에 기록됩니다.
